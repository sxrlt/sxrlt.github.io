安装Django

Posted on 2018-11-29

Django如何创建一个项目并进行访问：

准备工作：

- python-3.6.5环境（可自行下载安装）、virtualenv虚拟环境

下载virtualenv：

- 通过win+r快捷键打开运行输入cmd进入命令提示符
- 下载virtualenv
   pip install virtualenv  
  # 注意运行pip时要保证python的环境变量配置正确才能直接运行

创建虚拟环境：

- 创建一个文件夹env(名字可自拟)当作虚拟环境，位置可自己设置
- 切换到创建的文件夹目录下
- 创建一个名为djenv虚拟环境
  virtualenv --no-site-packages djenv  
- 切换到创建的虚拟环境目录下的Scripts文件加下
  cd djenv\Scripts
- 进入虚拟环境
  activate


- 查看虚拟环境已有的包
  pip list

下载django并创建django项目

- 下载django(这里我用的1.11版本)
  pip install django==1.11 
- 查看下载的django
  pip list    # 即可看到下载的django
- 新建一个文件夹workspace(名字可自拟)用于放项目代码
- 切换到创建的文件夹下（注意：在虚拟环境下）
- 创建第一个diango项目
  django-admin startproject day01
- 启动django项目
  python manage.py runserver

访问项目

- 在相应的浏览器输入ip地址120.0.0.1:8000即可访问自己的第一个项目
